    Tema 1. Introducció a la programació. Activitats

    Compendi d'activitats per al tema 1

    Diagrames de Flux

    Activitat 1. Fent zapping.

    Huí juga el Barça a les 21:00 però no se si el televisen en algun dels canals que tinc. Dissenya el diagrama de flux necessari per esbrinar si televisen o no i per quin canal és per poder veure-ho.

    Activitat 2. Recepta de la truita de creilla

    Fes un diagrama de flux amb els passos necessaris per fer una truita de creïlla (amb ceba)

    Actiivat 3. Com anar de Dénia a València utilitzant el transport públic.

    Imagina que et trobes a la plaça del convent de Dénia i vols anar a València utilitzant exclussivament el transport pùblic. Dissenya un algorisme amb la tècnica dels diagrames de flux per poder aconseguir-ho.

    Pseudocodi

    Activitat 4. De flux a psuedocodi

    Diseenya en pseudocodi les acitivtats de l'apartat de diagrames de flux

    Tema 2. Programació Estructurada.

    Activitat 1. Hola món.

    Com no tota primer activitat en un llenguatge de programació ha de ser el Hola món. A continuació podeu veure el codi necessari en Java per poder mostrar per terminal l'Hola món. 

    import java.util.*;

public class HolaMon {
    public static void main (String [] args) {
        System.out.println("Hola món");
    }
}

    Modifica el codi anterior per tal que diga "Hola Pablo" i a continuació també diga "Adéu Rocio".
